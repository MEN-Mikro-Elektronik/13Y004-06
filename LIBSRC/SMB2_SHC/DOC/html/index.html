SMB2SHC Documentation

(Shelf Controller API)

This document describes the SMB2SHC API to access the MEN Shelf Controller.
The Shelf Controller also called SHC can be controlled using the I2C/SMBus
interface.
The SMB2SHC API uses the SMBus interface from the SMB2_API which
provides e.g. ReadByteData and ReadBlockData to access the SMBus.


The main features of the Shelf controller API are:

  • Get the power supply status SMB2SHC_GetPSU_State()
  • Get the system temperature SMB2SHC_GetTemperature()
  • Get the fan status SMB2SHC_GetFAN_State()
  • Get the voltage levels SMB2SHC_GetVoltLevel()
  • Get the UPS charging state SMB2SHC_GetUPS_State()
  • Get the configuration data SMB2SHC_GetConf_Data()
  • ...


Calling example of a SMB2SHC function: 

  err = SMB2SHC_GetTemperature(u_int16 *value);
